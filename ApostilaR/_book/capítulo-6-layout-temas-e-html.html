Resumo Detalhado - Capítulo 6: Layout, Temas e HTML

Capítulo 9 Capítulo 6: Layout, Temas e HTML

Este capítulo trata de como controlar a aparência de um aplicativo Shiny usando layout, temas e elementos HTML. Os principais tópicos abordados são:

9.1 1. Layouts

O Shiny fornece diversas funções para organizar a interface dos aplicativos.

9.1.1 fluidPage()

Cria um layout fluido que se ajusta ao tamanho da tela do usuário.

ui <- fluidPage(
  titlePanel("Exemplo de layout fluido"),
  sidebarLayout(
    sidebarPanel(
      sliderInput("num", "Escolha um número:", 1, 100, 50)
    ),
    mainPanel(
      textOutput("resultado")
    )
  )
)

9.1.2 sidebarLayout()

Divide a tela em duas áreas: - sidebarPanel(): normalmente usada para inputs. - mainPanel(): usada para saídas (outputs).

9.1.3 fluidRow() e column()

Permitem criar layouts personalizados com múltiplas colunas:

ui <- fluidPage(
  fluidRow(
    column(4, "Coluna 1"),
    column(4, "Coluna 2"),
    column(4, "Coluna 3")
  )
)

Cada column() recebe uma largura entre 1 e 12.

9.2 2. Temas com {bslib}

A partir da versão mais recente do Shiny, é possível usar o pacote {bslib} para personalizar temas com Bootstrap.

9.2.1 Exemplo básico com tema:

library(bslib)

ui <- fluidPage(
  theme = bs_theme(bootswatch = "flatly"),
  titlePanel("App com tema Flatly"),
  sidebarLayout(
    sidebarPanel("Barra lateral"),
    mainPanel("Conteúdo principal")
  )
)

Você pode mudar cores, fontes e muito mais com bs_theme():

theme <- bs_theme(
  bg = "#002B36", fg = "#EEE8D5",
  primary = "#2AA198",
  base_font = font_google("Roboto")
)

9.3 3. HTML Customizado

O Shiny permite inserir HTML diretamente na interface.

9.3.1 Inserindo HTML diretamente

ui <- fluidPage(
  HTML("<h1 style='color: blue;'>Título Azul</h1>"),
  p("Parágrafo normal em Shiny")
)

9.3.2 Funções helpers

Shiny inclui funções para criar elementos HTML sem precisar escrever o HTML bruto: - tags$h1(), tags$p(), tags$a(), etc.

ui <- fluidPage(
  tags$h2("Título gerado com tags$h2"),
  tags$a(href = "https://rstudio.github.io/shiny/", "Documentação do Shiny")
)

9.4 4. Dependências de HTML

Você pode adicionar dependências de JavaScript e CSS com: - includeCSS("estilo.css") - includeScript("script.js") - includeHTML("arquivo.html")

9.5 5. Exercício Prático

Crie um app com: - Layout com 2 colunas. - Um tema da galeria do bootswatch. - Um título com tags$h1() e cor personalizada.

ui <- fluidPage(
  theme = bs_theme(bootswatch = "cosmo"),
  tags$h1("Meu App Estilizado", style = "color: darkred;"),
  fluidRow(
    column(6, "Coluna A"),
    column(6, "Coluna B")
  )
)
